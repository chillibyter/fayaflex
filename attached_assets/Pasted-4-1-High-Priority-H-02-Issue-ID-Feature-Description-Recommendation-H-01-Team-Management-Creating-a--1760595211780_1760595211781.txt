4.2. Medium Priority
Issue
ID
Feature Description Recommendation
M-01
Track
Activity
After submitting a
fitness activity, there is
no confirmation
message, and the form is
not cleared. This leaves
the user uncertain if the
submission was
successful.
Display a clear and immediate
confirmation message (e.g., a toast
notification or an inline message)
upon successful activity submission.
The form fields should also be reset
to allow for another entry.
M-02 Dashboard
There is a noticeable
delay for new activity
data to appear on the
dashboard, which can
cause confusion.
Investigate and optimize the data
synchronization process between the
activity logging and the dashboard
display to ensure data appears in
near real-time. If a delay is
unavoidable, consider adding a
visual indicator that the dashboard is
updating.
M-03 Dashboard
For new users, the
dashboard displays a
rank of "#0" and a "Top
10%" indicator, which is
misleading before any
activity is logged.
For new users with no activity data,
either hide the rank and percentage
indicators or replace them with a
welcoming message that guides the
user to log their first activity.
